## The first thing
This site uses GitHub Pages. You should be aware that GitHub Pages are mainly used for writing blogs or project wikis. The pages are static, meaning it displays the same content to every user. If you wanted to build a dynamic page (content change based on user actions or data retrieval from database), you avoid using GitHub Pages.
